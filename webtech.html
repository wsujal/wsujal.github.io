CSS Types & Syntax

CSS (Cascading Style Sheets) defines how HTML elements are displayed.

CSS Syntax

      property: value;
    }

Types of CSS

TypeDescription
InlineInside HTML element using style attribute
InternalInside <style> tag in head
ExternalSeparate .css file linked with link tag
<p style="color:red;">Inline CSS</p>
    
    <style>
    p { color: blue; }

Selectors — id vs class

id Selector

  • Unique element
  • Uses # symbol
  • Higher specificity

class Selector

  • Reusable
  • Uses . symbol
  • Multiple elements allowed
#header {
      color: white;
    }
    
    .box {
      color: orange;
    }

Other Selectors

* { margin:0; }
    p { color:blue; }
    div p { color:red; }
    a:hover { color:orange; }
